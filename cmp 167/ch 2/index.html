Write a program that asks the user for three doubles (firstDouble, secondDouble, thirdDouble), perform the following operations and output the results:

 

1.        resultOne = Add thirdDouble and secondDouble then divide the result by firstDouble

2.        resultTwo = Multiply thirdDouble by secondDouble and divide the result by the sum of firstDouble and secondDouble

 

Your prompt to the user to enter the doubles must be:

Enter firstDouble:

Enter secondDouble:

Enter thirdDouble:

 

Your output must be of the format:

First Result = resultOne

Second Result = resultTwo

 

Please make sure to end each line of output with a newline.
